# 认领说明
翻译前，请在 issue 处认领

认领格式：此章节由我翻译，翻译时间为 2018.10.19-2018.10.21

如果本人认领，然后发现没有时间翻译，可以取消认领

取消认领格式：此章节我暂时没有时间翻译，请其他同学协助代为翻译，谢谢！

翻译完成，到领取的 issue 中回复：此章节我已翻译完成。见 PR #1 等待维护人员进行后续处理即可。

# 校译说明
校译认领与前面“认领说明”一致操作相同。

采用交叉检查法，每个翻译志愿者在完成自己的任务之后，需要完成同等任务量的其他译者的校译工作。

# 工作量说明
